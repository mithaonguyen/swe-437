# Unit Conversion Program

**Author:** Mimi Nguyen

**Description:** Java-based unit conversion program that allows users to convert between various units including temperature, distance, volume, weight, time, and speed.

**Files Included:**
* ```conversion.java``` - Main program with all conversion methods and UI
* ```conversiontest.java``` - JUnit test file for all conversion functions
* ```README.md``` - instructions for compiling and running the code

**How to Compile and Run**
1. Open a terminal in the project folder and enter the command ```javac conversion.java``` to compile the program.
2. Run the program by entering the command ```java conversion```. Follow the menu prompts to select a conversion and enter input values.
